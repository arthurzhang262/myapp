# 环境变量配置示例
# 请将此文件复制为 .env.local 并填入实际的值

# 数据库配置
DB_HOST=127.0.0.1
DB_USER=root
DB_PASSWORD=your_password_here
DB_NAME=test_db
DB_PORT=3306

# 应用配置
NEXT_PUBLIC_APP_URL=http://localhost:3000

# 重要提示：
# 1. 不要将包含真实密码的 .env.local 文件提交到版本控制系统
# 2. 确保 MySQL 服务正在运行
# 3. 确保数据库和表已创建（参考 database-init.sql）
# 4. 确保用户有访问数据库的权限

# 创建 .env.local 文件的步骤：
# 1. 在项目根目录创建 .env.local 文件
# 2. 复制上面的配置内容
# 3. 将 your_password_here 替换为实际的数据库密码
# 4. 保存文件并重启开发服务器
